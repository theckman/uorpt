UOrpt
=============

CONTRIBUTING
------------
Thank you for your interest in contributing back to the UOrpt project. To
contribute back the following steps will do:

* Please fork this repository and make your changes on a feature branch
(i.e., don't commit on the master branch)

* Please be sure to write any applicable tests, or adjust the tests as needed

* In your commit message use full sentences to properly explain the changeset
and why you feel it would be a beneficial improvement. If there are any
breaking API changes please mention those as well.

* Lastly, open a PR on the project from your feature branch
